About

CDev is a functional (no classes), staticly typed programing language that's designed to be used for teaching purpuses. CDev is writen in C#.


Printing and Input

To print something or get user input in CDev the 2 pre-defined functions Write() and Read().

Write("Enter your name");

name = Read();

Write("Hello ",  name);

Write

Automatically the Write() function will create a new line when it finishes. As the example above shows you it is possible to pass multipile arguments to it that will be printed in the same line.

Read

The Read() function will return what the user inputs.


Variable declaration

In CDev there are three types of variable declerations and this look somehing like this:

def i: int = 10;  //typed

def i = "Hello World";  //automatic

def i;  //null


1. Typed

Declearing a variable this way will require a type to be specified after the name of the variable. This will create a variable that has a specific type, in this case, int. that types can be the following:

    - int: Intiger number
    - float: Decimal number
    - bool: True or False
    - string: Word, sentence or character
    - any: Can be any of the above
Note that once you declear a variable you can not change its type. Also note that passing a value is not needed.


2. Automatic

Declearing a variable this way will require a value be given after the variable name. This will work the same way as a typed declaration except the language will automatically dicite the type of the variable.


3. Null

Declearing a variable this way will require that neither a value or a type shall be specified. Creating a variable in this way will give the variable the type null. When assigning a value to it, it will automatically change its type to the one of the value assignned to it.


Arrays, Lists and Dictionaries

Arrays

To create and array it is requred that after the type a, in brakets, the size is given. Like this:
def: int [ 10 ] i;

Lists

 If a size is not spesifed then a List will be created. A list is a dynamic array that will change it's size depending on how much nontent is in it. This is how it can be done:
def i: string [];

Dictionaries

 A dictionary has 2 parts, value and code. Each code represents one velue, like a hash map. The syntax to declear dictionaries is different then the rest. It is as following:
def i: dictionary [ float ];
This will create a dictionary that stores float numbers. But its code can be anything. For example.
i [ "Age" ] = 19.7;
In this case the code is "Age" and the value is 19.7.Now getting the age is a simple as this:
Write(i [ "Age" ]);

If there's a variable named i already it will call it j, if j is taken then it will call it k. It is possible to give a name to the variable after the numbner of itterations like so:

An initial value can be assinged to it like so:

Instead of the number of itewrations, at the start, a list can be passed of value that thebe assinged to the variable. This can be done like so:

Function Declaration

Declearing a function in CDev is very similar to a declearing a varialbe. The syntax is the following:

func ReturnHelloWorld(): string{
 return "Hello World" ;
}
The type of the function must be one of the following depending on what the functions returns:
    - int: Returns an intiger number
    - float: Returns a decimal number
    - string: Returns a sentence, word or character
    - bool: Returns True or False
    - any: Returns any of the above
    - void: Doesn't return anything
Unlike variable declaration, a type must be given.
